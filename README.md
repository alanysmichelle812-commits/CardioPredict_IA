# 🩺 CardioPredict S.A. - Plataforma de Predicción de Riesgo Cardiovascular IA

[![Estado del Despliegue](https://img.shields.io/badge/Streamlit_App-Online-brightgreen?style=for-the-badge&logo=streamlit)](https://cardiopredictia-8lbcpmlbjfcetiyzrxuhqf.streamlit.app/)
[![Algoritmo Principal](https://img.shields.io/badge/Algoritmo-XGBoost-blue?style=for-the-badge&logo=kaggle)](https://xgboost.ai/)
[![Precisión del Modelo](https://img.shields.io/badge/Precisión-81.97%25-red?style=for-the-badge)](https://cardiopredictia-8lbcpmlbjfcetiyzrxuhqf.streamlit.app/)
[![Licencia](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](LICENSE)

Este proyecto, desarrollado como solución de Ingeniería Biomédica, presenta la plataforma **CardioPredict S.A.** para la evaluación predictiva de enfermedades cardíacas. Utiliza Machine Learning avanzado para ofrecer un diagnóstico rápido y fiable.

## 🛠️ 1. Componentes Clave

| Componente | Rol | Herramienta |
| :--- | :--- | :--- |
| **Modelado** | Clasificación Binaria de Riesgo | XGBoost (Python) |
| **Precisión** | Métrica de Rendimiento del Modelo | **81.97%** |
| **Frontend** | Interfaz de Usuario Web | Streamlit |
| **Archivos Clave** | Lógica de la aplicación y dependencias | `app.py`, `cardio_model.joblib`, `requirements.txt` |

## 💰 2. Modelo de Negocio (SaaS B2B LatAm)

La estrategia de precios se basa en el alto valor generado por la precisión del **81.97%** en la mitigación del riesgo clínico. 

| Plan | Enfoque | Tarifa |
| :--- | :--- | :--- |
| **Micro** | Consultorios de bajo volumen | **$0.75 USD / predicción** |
| **Básico** | Clínicas pequeñas | **$99 USD / mes** (**Hasta 150 análisis**) |
| **Empresarial** | Hospitales y aseguradoras | **$399 USD / mes** (Análisis ilimitados, Acceso API) |

## 🧑‍💻 3. Replicar el Proyecto Localmente

Para iniciar la aplicación en su máquina:

1.  **Clonar:** `git clone https://github.com/alanysmichelle812-commits/CardioPredict_IA.git`
2.  **Dependencias:** `pip install -r requirements.txt` (dentro de un entorno virtual).
3.  **Ejecutar:** `streamlit run app.py`
